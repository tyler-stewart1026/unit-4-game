Welcome to Gem Collector!

Here you will be assigned a random number when the game starts.

Below are four crystals. Each crystal has a value which is hidden from you until you click on it. The game is won when you click on crystals whose values add up to the random number. You will lose the game, however, if the values you're selecting exceed the value of the random number.

Everytime the game resets upon a win or loss, the values of the crystals change and a new number is generated.
